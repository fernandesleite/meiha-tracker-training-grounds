# Meiha - Android Movie Tracker
Android app to track and search Movies from [The Movie Database API](https://developers.themoviedb.org) using

  - Kotlin
  - MVVM Pattern
  - Android Architecture Components (LiveData, Room, ViewModel)
  - RecyclerView
  - Retrofit
  - Glide
  - Gson
  - Moshi
